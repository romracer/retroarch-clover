=== Mupen64Plus core for RetroArch ===

** Formerly the GLupeN64 core **

This module will add support for Nintendo 64 games to RetroArch

Available executables and arguments:
- /bin/n64 <rom> <clover_args>
  runs "mupen64plus" core
